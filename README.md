# 💻 Ingeniería de Software II

Repositorio correspondiente a la materia **Ingeniería de Software II** de **2° año** de la *Tecnicatura Superior en Desarrollo de Software* del **Instituto Superior de Profesorado N° 20**.

---

## 📚 Descripción

En este repositorio se desarrollarán distintos trabajos prácticos, actividades y proyectos relacionados con el diseño y desarrollo de aplicaciones web, aplicando conceptos de ingeniería de software.

El objetivo principal es fortalecer habilidades en desarrollo frontend, buenas prácticas de programación y organización de proyectos.

---

## 🚀 Tecnologías utilizadas

Durante el cursado se trabajará con las siguientes tecnologías:

- 🧱 HTML5  
- 🎨 CSS3  
- ⚙️ JavaScript  
- 📦 Bootstrap  

---

## 📁 Estructura del proyecto

El repositorio se organizará de la siguiente manera:

```
/proyecto
  ├── index.html
  ├── css/
  │     └── styles.css
  ├── js/
  │     └── script.js
  ├── assets/
  │     └── (imágenes, íconos, etc.)
  └── README.md
```

---

## 🛠️ Objetivos de la materia

- Comprender el proceso de desarrollo de software  
- Aplicar metodologías y buenas prácticas  
- Desarrollar interfaces web modernas y responsivas  
- Integrar diferentes tecnologías del lado del cliente  

---

## 📌 Requisitos

Para ejecutar los proyectos solo necesitás:

- Un navegador web actualizado (Chrome, Edge, Firefox, etc.)  
- Un editor de código (VS Code recomendado)  

---

## ▶️ Cómo usar este repositorio

1. Clonar el repositorio:
```bash
git clone https://github.com/tu-usuario/tu-repositorio.git
```

2. Abrir el proyecto en tu editor de código  

3. Ejecutar el archivo `index.html` en tu navegador
